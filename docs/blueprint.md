# **App Name**: NestFind

## Core Features:

- Property Listings: Display properties with photos and detailed information, supporting both sales and rentals.
- Advanced Search: Allow users to search for properties by location, price range, and other criteria, across multiple vendors and agencies.
- Property Details: Dedicated pages for each property with comprehensive details, photos, and agency information.
- Multi-Vendor Platform: Enable real estate agencies to list properties, manage listings, and connect with potential clients.
- Property Collections: Enable users to make and save properties in a private collection.
- AI-Powered Search: Implement a search feature allowing users to find properties using natural language queries. This feature uses an LLM as a tool to understand the customers requirements.
- AI-Driven Recommendations: Use user's past behavior and entered requirements, to tailor property suggestions. An LLM serves as a tool for determining property needs.

## Style Guidelines:

- Primary color: Professional blue (#3498DB) to convey trust and reliability.
- Background color: Clean white (#F5F5F5) for a modern and open feel.
- Accent color: Light blue (#79ABE6) for interactive elements and highlights, offering a soft contrast.
- Body font: 'PT Sans', a humanist sans-serif for readability and modern appeal.
- Headline font: 'Space Grotesk', a sans-serif for impactful headlines, complementing the body text with a techy edge.
- Use modern, minimalist icons related to property types and features.
- Grid-based layout for property listings and detailed property pages.
